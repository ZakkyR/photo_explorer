# やりたいこと
- 画像の整理を行うWindowsアプリ

# 作成の理由
作業用画像フォルダにたくさんの画像をおいて、エクスプローラーを使って、ドキュメント作成で使う画像を選んでいたが、以下の面で困っていた

- 画像表示が小さいのでいちいち拡大が必要
  - 同じ構図の写真で比較するのがやりづらい
- どれがどの写真かわからない
  - リネームするのは同じ構図を複数枚撮ったときに面倒
  - フォルダ分けしちゃうとフォルダをいちいち開くのが面倒
- 画像フォルダがあちこちあったりしてマージが面倒

# 機能
- 画像のフォルダを追加
- 追加したフォルダの中にある画像を表示
- 画像にタグ付けができる
- タグで絞り込みができる
- 一覧からドラッグアンドドロップで他のアプリに持っていける
  - エクスプローラーと同じ操作感がほしい
  - 一番やりたいことはAffinityに貼り付けがやりたい
- ダブルクリックでプレビュー
  - 左右キーで画像切替
  - Windowsのビューアのような形
    - Windows標準で対応できるならそれでもOK
- フォルダに画像が追加されたら自動的に追加される
  - フォルダの中身を参照しているぐらいスムーズだといい
- アルバム
  - 複数フォルダをまとめて表示
  - 一つのフォルダみたいに扱える
  - 各アルバムの仕様はフォルダのときと同じ
